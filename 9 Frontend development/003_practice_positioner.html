Types of Positioners (this is a fixed element)

  • Static: Default html positioning
  • Absolute: Relative to the nearest positioned ancestor
  • Relative: Elements relative to its initial position
  • Fixed: Elements that wont move even if the page changes
  • Sticky: Intermediate between Relative and fixed
